== Description ==

Events plugin integrates well with your WordPress site. This plug-in allows you to create events that have the same functionality as posts, while adding further features that allow you to manage your events by adding venue, location, start and end date. It will allow to export past and upcoming event. It will display events and filer in frontend by using [events] shortcode.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `event.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[events]` shortcode in your templates/page
